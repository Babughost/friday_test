Git command list

01. git init: 새로운 Git 저장소를 초기화합니다.
02. git clone: 원격 저장소의 내용을 로컬 컴퓨터로 복제합니다.
03. git add: 변경된 파일을 스테이징 영역에 추가합니다.
04. git commit: 스테이징 영역에 있는 파일들을 커밋하여 변경 내용을 저장소에 반영합니다.
05. git push: 로컬 저장소에 있는 변경 내용을 원격 저장소에 업로드합니다.
06. git pull: 원격 저장소에서 변경된 내용을 로컬 저장소로 가져옵니다.
07. git merge: 다른 브랜치의 변경 내용을 현재 브랜치로 병합합니다.
08. git branch: 새로운 브랜치를 만들거나 현재 브랜치를 확인합니다.
09. git checkout: 다른 브랜치로 이동하거나 특정 커밋의 내용을 확인합니다.
10. git log: 저장소의 커밋 로그를 확인합니다.
